How to Pair Program

What Is Pair Programming?

  • Working together to find a solution to coding issues
  • The ability to work remotely with others

What To Do.

  • Link up online
  • Check in
  • Discuss the work or any issues
  • Start doing the work

Why Do It?

  • It's effective
  • Things are done quicker
  • Raises productivity so that not just one person is trying to solve a problem
